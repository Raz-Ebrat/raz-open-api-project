Richmond & Kandahar Weather Comparison

A colorful Open API project using real-time weather and time data

About

This is my API Project built with the Open-Meteo API. This project compares real-time weather and time information for Richmond, Virginia and Kandahar, Afghanistan.

I wanted to make the project more meaningful by comparing two places from different parts of the world and presenting the data in a clean, visually appealing way.
